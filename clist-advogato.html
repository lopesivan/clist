• C++ template library of circular lists

  • sorting, finding, etc. over arbitrary circular ranges

  • allows const-qualified element types

  • completely integrated functor system
    • compare elements when sorting
    • compare elements when searching
    • perform unary analysis when searching
    • convert elements from another list to insert
    • bind to lists for output conversions
    • detects functions, pointers, and objects

  • sort using external reference data

  • sort external reference data based on the list

  • create shadows of original lists without changing the original
    • manipulate element order
    • sort the elements
    • remove elements
    • can be bound to the original list

  • optional virtual array storage
    • stores pointers to dynamically-allocated elements
    • compile-time constant
    • drastically increases sorting speed with large elements
    • provides stable pointers
    • can be selected on a per-class basis
    • can be completely enabled or disabled
    • automatic for large element types by default
    • 100% transparent
    • uses the same allocator template for individual elements

  • inverted sorting
    • can be used to unsort external data
    • can be used to select element order

  • maintained consistently since 2003, released since 2005
